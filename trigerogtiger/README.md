# Triger'n og Tiger'n – Blogg

Gjenopprettet blogg fra trigerogtiger.blogg.no (2008–2014), hentet fra web.archive.org.

## Innhold

Bloggen dekker familielivet til Finn Magne og Tina med sine seks barn, inkludert:
- Reiser til Sardinia, Madrid, Amsterdam, Bodø, Vestlandet og Finnskogen
- Korpslivet i Tjodalyng Skolekorps og Nordsøfestivalen i Danmark
- Hverdagsliv, katter, tvillinger og baking

## Teknisk

Enkel statisk HTML-side, hostet på Vercel.

## Deployment

Koblet til Vercel – alle push til `main` deployes automatisk.
